# Change: Unify the Liangshanbobo public brand

## Why

The public GitHub profile, personal website, and flagship knowledge product
currently mix placeholders, the old `LIANG.0417` identity, and the deprecated
`L41 Context` product name. Together they act as one public geek card, so the
identity and URLs need one explicit source of truth.

## What changes

- Use `Liangshanbobo` as the public personal brand.
- Keep `@liang0417` as the stable GitHub login and profile repository name.
- Rename the personal website repository to `liangshanbobo`.
- Rename `L41 Context` to `Shanbo Context`, including its repository,
  package names, UI, documentation, and public links.
- Update the GitHub profile README to point at both renamed repositories.
- Remove public placeholders such as `YOUR.NAME`, generic GitHub links, and
  the unused email placeholder.

## Impact

- Personal website repository: identity copy, metadata, links, package name,
  GitHub Pages base path, docs, and licensing attribution.
- GitHub profile repository: featured project links and names.
- Flagship product repository: product name, repository URL, Python import
  package, frontend package, documentation, and examples.
- GitHub repository names and the profile website URL.
